## 使用方式

### ROS 1 真机导航（默认使用项目 maps/）

```bash
roslaunch turtlebot3_delivery delivery_fsm.launch
# 或指定：map_file:=$(pwd)/maps/restaurant_map.yaml
```

### ROS 1 单独导航（含 AMCL）

```bash
roslaunch turtlebot3_delivery navigation.launch
```

### 标定送餐目标点

1. 先启动 `navigation.launch`（带地图）
2. 在 RViz 中用 **Publish Point** 或运行 `mark_goal_points.py` 标出厨房、桌子位置
3. 坐标会自动写入 `fsm_map_config.yaml` 的 `locations`

## 相关配置

- `catkin_ws/src/turtlebot3_delivery/config/fsm_map_config.yaml` — 地图路径、目标点、区域
- `catkin_ws/src/turtlebot3_delivery/config/fsm_params.yaml` — FSM 参数、locations 备份
